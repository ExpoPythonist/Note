# Backend

## Install dependencies 

```bash
pip install -r requirements.txt
```

**Running locally:**

```bash
python manage.py runserver
```

# Frontend

## Install dependencies 

```bash
npm install 
```

**Running locally:**

```bash
npm run dev
```
